## 目次構成

### 基礎編（第1部〜第3部）
- **入門編**: GitHubとは、アカウント設定、リポジトリの基本
- **日常操作編**: Git操作、ブランチ、PR、コードレビュー
- **WebUI操作編**: ブラウザでの編集、github.dev、デプロイ

### 運用編（第4部〜第6部）
- **運用フロー編**: ブランチ戦略、命名規則、リリースフロー
- **ルール設定編**: ブランチ保護、Rulesets、CODEOWNERS
- **Issue・プロジェクト管理編**: Issues、Projects、Discussions

### 機能編（第7部〜第9部）
- **GitHub Actions編**: CI/CD、ワークフロー自動化
- **セキュリティ編**: Dependabot、Code Scanning
- **GitHub Copilot編**: AI支援の活用（7章の詳細解説）

### 上級編（第10部〜第13部）
- **組織・チーム管理編**: Organization、Enterprise
- **その他の機能編**: Pages、Packages、Codespaces
- **API・CLI編**: gh コマンド、REST/GraphQL API
- **トラブルシューティング編**: よくある問題と解決法
